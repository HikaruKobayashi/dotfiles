# dotfiles
Easily build an environment with dotfiles.

## Development environment
|              |  My Environment  　　|
|:---:|:---:|
|OS|Mac OS|
|shell|fish|
|Terminal|iTerm2|

## How to Set up
`$ git pull https://github.com/HikaruKobayashi/dotfiles.git`

### fish
`$ cd dotfiles`
`$ sh setup.sh`

### Visual Studio Code
`$ cd vscode`
`$ sh vscode.sh`
